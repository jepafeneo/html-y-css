# HTML Sitio Web

Hacer un repositorio y trabajar en la rama dev

1. Pagina de inicio index.html

Consigna:

Crear un HTML que represente una página personal o institucional.

Debe tener:

  <header> con nombre
  <nav> con al menos 3 enlaces dentro de una lista
  <main> con 2 secciones
  <footer> con form newsletter

Tabla de horarios / precios / materias.

Prohibido: <div> para todo.

2. Crear una pagina Artículos (blog, noticias, novedades)

Consigna:

Armar varios artículos de blog.

Debe incluir:

  <article>
  <h2> a <h3>
  <p>, <strong>, <em>
  Una cita (<blockquote>)

3. Galería

Consigna:

Crear una pagina "Galería".

Debe incluir:

  <figure> + <figcaption>
  <img> con alt

4. Una pagina de Contacto

Consigna:

    Formulario de contacto.

Debe incluir:

    <label> bien asociado
    <input> (text, email)
    <textarea>
    <button>

Al menos un required
