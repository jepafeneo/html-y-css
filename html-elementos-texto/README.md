# HTML - Elementos de texto

## Encabezados

```html
<h1>Encabezado principal</h1>
<h2>Subtítulo</h2>
<h3>Subsección</h3>
<h4>Subnivel</h4>
<h5>Subnivel menor</h5>
<h6>El más pequeño</h6>
```

### Ejemplo 

```html
<h1>Lenguajes</h1>

<h2>HTML</h2>

<h2>CSS</h2>

<h2>JavaScript</h2>
<h3>Framework</h3>
<h3>Librerias</h3>
```

## Parrafos

```html
<p><strong>HTML</strong> (Lenguaje de Marcado de Hipertexto) es el código fundamental que estructura el contenido de una página web, usando etiquetas para organizar textos, imágenes, videos y enlaces, definiendo así la base de cualquier sitio web, a menudo combinado con <em>CSS</em> para el estilo y JavaScript para la interactividad.</p>
```
